# Information 
 - This plugin allows players to store their money in a bank.
# Features 
 - Withdraw Money into Bank
 - Deposit Money from Bank
 - Transfer Money to Other Players Bank Accounts
# Command
- /bank
# Important
- This plugin Requires EconomyAPI by OneBone
